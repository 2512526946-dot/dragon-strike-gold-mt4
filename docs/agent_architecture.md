# 多智能体架构规划

巨龙出击未来采用多智能体分工 + 智能体经理汇总。当前阶段不开发智能体代码，不接入机器学习，也不让智能体输出真实交易建议。

## 未来智能体角色

1. 数据质量智能体
   检查 MT4 文件、metadata、freshness 和 DataQualityGate。

2. 黄金多周期分析智能体
   分析 D1 / H4 / H1 / M15 的趋势、震荡、关键位和假突破风险。

3. 市场环境智能体
   判断波动率、点差、流动性、时间段和新闻风险。

4. 风控智能体
   判断单笔 1% 风险、单日 3% 亏损、不隔夜、连续亏损等规则。

5. 仓位智能体
   根据账户权益、止损距离、tick_value、min_lot、lot_step、点差和 10x 压力测试给出建议仓位。

6. 交易纪律智能体
   识别追单、频繁交易、报复交易、违反系统建议、超仓位等行为。

7. 复盘智能体
   分析每次机会、纸面结果、真实执行结果、错误归因和纪律评分。

8. 安全合规智能体
   确保系统不自动交易，不绕过 DataQualityGate、RiskGate、PositionGate、GoLiveGate，不输出真实交易指令。

9. 智能体经理 / Agent Manager
   汇总所有智能体意见，输出最终动作。

## 智能体经理输出边界

智能体经理最终只允许输出：

- 观望
- 允许人工决策
- 禁止交易

## 一票否决权

拥有一票否决权的智能体：

- 数据质量智能体
- 风控智能体
- 仓位智能体
- 隔夜规则智能体
- 安全合规智能体

## 核心原则

- 行情智能体认为有机会，不等于可以交易。
- 方向判断正确，不等于可以开仓。
- 可以开仓，不等于可以重仓。
- 智能体可以解释原因，但不能绕过数据质量、风控、仓位和安全边界。

## 当前阶段边界

当前阶段不实现智能体代码，不实现机器学习，不实现自动交易。所有智能体内容都是未来架构规划。
