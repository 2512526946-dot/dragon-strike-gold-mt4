# 架构决策

本文记录巨龙出击当前确认的产品边界和架构方向。所有内容仅用于开发规划，不是交易建议。

## 核心定位

巨龙出击是黄金 XAUUSD 专用的交易辅助、风控、仓位、复盘和训练系统。它关注 MT4 适配、数据质量、人工决策、纪律纠偏和复盘训练，不定位为通用量化框架。

当前阶段的所有功能都属于只读、观察、复盘、架构规划或数据质量检查。当前系统不自动交易，不提供真实交易建议。

## 产品形态

最终形态为：

```text
本地网页控制台 + MT4 只读桥接 EA + 人工下单
```

不是：

- 手机 App 优先
- 完整嵌入 MT4 的复杂插件
- 自动交易机器人

MT4 负责行情、K 线、品种规格、账户快照和人工下单入口。巨龙出击负责数据质量检查、黄金多周期分析、风险闸门、仓位建议、观察信号、复盘、智能体解释和纪律监督。

## 只读桥接边界

只读 MT4 文件桥接用于导出数据，不用于执行交易。

当前规划中的四类文件：

- `live_tick.json`：tick 快照
- `latest_bars.json`：M15 / H1 / H4 / D1 K 线快照
- `symbol_spec.json`：品种规格、点值、手数、保证金
- `account_snapshot.json`：账户权益、保证金、日内亏损限制、未来风控基础

四个文件只是数据输入，不等于系统允许交易。

## QuantDinger / 量化丁格借鉴边界

巨龙出击不照搬 QuantDinger，不复制其代码和策略。

可借鉴的系统思路：

- 系统分层
- 数据输入
- 数据质量检查
- 信号生成
- 风控过滤
- 仓位管理
- 模拟执行
- 复盘记录
- 策略改进
- Champion / Challenger 机制

巨龙出击的差异化重点：

- 黄金 XAUUSD 专注
- TradeMax Global MT4 适配
- H4 / H1 / M15 周期
- 不隔夜规则
- 单笔 1% 风险
- 单日 3% 亏损限制
- 仓位由小到大
- 人工决策
- 复盘训练
- 新手纪律纠偏

## 当前阶段禁止开发

当前阶段不得开发真实 MT4 接入、真实行情接入、真实账户读取、交易策略、风控计算、仓位计算、GoLiveGate、AutoTradeGate、智能体代码、机器学习或自动交易。

OrderSend、OrderClose、OrderModify、OrderDelete 只能出现在安全边界说明中，不能作为当前功能或可执行逻辑出现。

## MT4 模拟账号训练方向

未来可以规划 MT4 Demo Account / 模拟账号训练方向，但必须遵守 demo-only、只读优先、半自动确认优先、自动模拟训练后置且默认关闭。

未来执行链路必须经过：

```text
智能体交易计划
  -> DataQualityGate
  -> RiskGate
  -> PositionSizing
  -> ExecutionGate
  -> 用户确认或后期 AutoDemoTrainingMode
  -> MT4 EA 在模拟账号执行
  -> ExecutionAuditLog
```

智能体不是执行者，不能绕过风控。RiskGate 和 ExecutionGate 均有否决权。EA 只执行被批准的命令。第一阶段不允许实盘，不保存账号密码或登录凭证。
