# 界面布局原则

巨龙出击未来界面定位为黄金交易驾驶舱，不是炫酷量化大屏。

## 信息优先级

核心原则：

1. 先告诉用户能不能做。
2. 再告诉用户为什么。
3. 再告诉用户最多能做多少。
4. 最后才展示行情细节。

第一屏优先级：

1. 今日最终动作
2. DataQualityGate
3. 今日最大亏损限制
4. 单笔最大风险
5. 隔夜规则
6. 仓位建议
7. 禁止原因
8. 智能体经理结论

## 允许的核心动作

页面核心动作只允许三种：

- 观望
- 允许人工决策
- 禁止交易

禁止使用以下文案：

- 立即买入
- 立即卖出
- 强烈做多
- 强烈做空
- 自动开仓
- 自动下单

这些禁止文案只能作为文档中的安全边界说明出现，不能作为界面动作或当前系统输出。

## 第一屏建议布局

顶部状态栏：

- MT4 连接状态
- 数据延迟
- 当前模式
- 自动交易关闭
- GoLiveGate 状态

第一行核心卡片：

- 今日最终动作
- DataQualityGate
- 今日最大亏损限制
- 单笔最大风险
- 隔夜规则

第二行分析区：

- 黄金多周期状态 D1 / H4 / H1 / M15
- 关键价格区域
- 当前市场环境

第三行决策区：

- 仓位建议
- 禁止原因
- 智能体解释

底部：

- 最近观察信号
- 复盘提醒
- 系统提醒

## 颜色原则

- 绿色：通过 / 正常
- 黄色：观望 / 等待 / 注意
- 红色：禁止 / 风险 / 阻断
- 蓝色：只读 / 信息
- 灰色：未启用 / 未连接

## 界面模式

后续支持两种界面模式：

- 新手模式：解释详细，适合学习和训练。
- 实战模式：只显示关键结论、风险、仓位和禁止原因。

## 当前阶段边界

当前前端页面仍是开发预览。它可以显示 Mock 数据、Placeholder 信号和 DataQualityGate 相关状态，但不展示真实交易建议，不自动下单，不连接真实账户。
